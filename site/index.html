bx-bluetooth

Browser Printer Harness

This static test page runs a Matchbox WebAssembly VM, executes a BoxLang harness script, and uses the `bx-bluetooth` module to validate printer discovery, GATT selection, and a simple TSPL write from a secure origin.

Printer Settings

Use Chrome or another Chromium-based browser on Android. Web Bluetooth requires HTTPS and a user gesture. Device selection is filtered by name prefix only; the service list is used as an access allowlist during GATT discovery by the BoxLang harness.
